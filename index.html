Polynomial Expressions

Project Overview

This project is a visual presentation of polynomial expressions, created by Bryan Moreno using pure HTML and CSS. It introduces different types of polynomials — monomials, binomials, trinomials, and general polynomials — with clear examples and notes. The layout mimics a formatted math reference sheet and is styled to be simple, readable, and educational for viewers learning algebraic fundamentals.

Techniques Used
  • Basic HTML structure for semantic clarity
  • <pre> tags for preformatted alignment of math-related content
  • Superscript and subscript for proper mathematical notation (e.g., x2, Q1)
  • Internal CSS for custom fonts, background color, and text styling
Project Purpose

The purpose of this project was to demonstrate how algebraic concepts can be represented on the web using HTML and CSS only — no scripts or libraries. It serves as a practice in both educational content formatting and in using preformatted blocks for aligned textual data.

Highlights
  • Categorizes polynomials based on the number of terms
  • Clean typographic presentation for academic use
  • Use of math notation with superscript/subscript HTML tags
  • Educational theme with creative layout

Project Overview

Techniques Used
  • HTML structure with headings and paragraphs
  • Links with dynamic states (hover, active, visited) and CSS animations
  • Internal CSS for basic styling and link interactions (color changes and blinking animation)
Project Purpose

The goal is to create a dynamic article layout with functional hyperlinks and styled content. It also demonstrates how CSS animations and transitions can be applied to elements for interactive design.

Highlights
  • Interactive hyperlinks with multiple states (visited, active, hover)
  • CSS animations (blinking effect)
  • Use of internal CSS for styling and animations
HTML
<!DOCTYPE html>
<html lang="en">
<head>
    <title>Hatsune Miku in Fortnite by Bryan Moreno</title>
    <link rel="stylesheet" type="text/css" href="ArticleLinks.css">
    <style>
        body {
            font-family: Tahoma;
        }
        a:link { color: red; text-decoration: underline; }
        a:visited { color: yellow; text-decoration: line-through; }
        a:active { color: green; text-decoration: overline; }
        a:hover { color: blue; animation: blink 1s step-start infinite; }
        @keyframes blink { 50% { opacity: 0; } }
    </style>
</head>

function getResult() {
    var num = parseInt(document.getElementById("numInput").value);
    var result1 = document.getElementById("result1");
    var result2 = document.getElementById("result2");

    if (num % 2 === 0) {
        result1.innerHTML = "THE NUMBER IS EVEN";
    } else {
        result1.innerHTML = "THE NUMBER IS ODD";
    }
    if (num <= 1) {
        result2.innerHTML = "INVALID";
    } else {
        var count = 0;
        for (var i = 1; i <= num; i++) {
            if (num % i === 0) {
                count++;
            }
        }
        
        if (count <= 2) {
            result2.innerHTML = "THE NUMBER IS PRIME";
        } else {
            result2.innerHTML = "THE NUMBER IS COMPOSITE";
        }
    }
}

function multiplication() {
    let rows = prompt("Enter number of rows:");
    let columns = prompt("Enter number of columns:");
    
    let table = "";
    
    for (let i = 1; i <= rows; i++) {
        table += "";
        for (let j = 1; j <= columns; j++) {
            table += "";
        }
        table += "";
    }
    
    table += "
" + (i * j) + "
";
    
    document.getElementById("multipy").innerHTML = table;
}
